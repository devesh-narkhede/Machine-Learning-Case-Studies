# CASE STUDY
# Ball Classification Using Decision Tree

------------------------------------------------------------

## Case Study Overview

This case study demonstrates the implementation of a 
Decision Tree Classification model to classify balls 
as either Tennis Ball or Cricket Ball.

The classification is based on physical characteristics 
such as weight and surface type.

------------------------------------------------------------

## Machine Learning Details

- **Machine Learning Type:** Supervised Learning  
- **Problem Type:** Classification  
- **Algorithm Used:** Decision Tree Classifier  

------------------------------------------------------------

## Dataset Details

Total Records: 15  

### Independent Variables (Features):

1. **Weight (in grams)**
2. **Surface Type**
   - Rough = 1
   - Smooth = 0

### Dependent Variable (Target):

- Tennis Ball = 1  
- Cricket Ball = 2  

------------------------------------------------------------

## Tools & Technologies Used

- **Python** – Programming language used for implementation.
- **Scikit-learn** – Used to build and train the Decision Tree model.
